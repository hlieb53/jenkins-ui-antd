## Usage

### Use bash

```bash
$ git clone git@github.com:jetlinks/jetlinks-ui-antd.git
$ cd <your-project-name>
$ yarn
$ yarn start:dev
```

`注意：请不要使用npm安装和启动项目`

More instructions at [](https://doc.jetlinks.cn).
